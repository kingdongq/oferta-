# Smart Poster Generator

AI驱动的智能海报生成器 - 基于 QROfertas 的增强版本

## 核心功能

- 🤖 **AI Agent 理解产品需求**
- 🖼️ **自动搜索产品图片**
- 🎨 **智能匹配模板**
- 📱 **自动生成多种格式海报**

## 快速开始

### 环境要求

- Python 3.9+
- Node.js 18+
- Claude API Key

### 安装

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 运行

```bash
# 启动后端
cd backend
python app.py

# 启动前端
cd frontend
npm run dev
```

## 技术栈

- **前端**: React + Fabric.js
- **后端**: Python FastAPI
- **AI**: Claude API
- **图片搜索**: Google Custom Search API
- **存储**: Cloudinary
