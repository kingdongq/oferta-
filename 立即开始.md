## 📝 步骤2：记录配置信息

创建文件 `环境变量.txt` 记录所有凭证：

```env
# Supabase
DATABASE_URL=postgresql://...
POSTGRES_PASSWORD=...

# Stripe
STRIPE_PUBLIC_KEY=pk_test_...
STRIPE_SECRET_KEY=sk_test_...

# Cloudinary
CLOUDINARY_URL=cloudinary://...

# Vercel (自动配置)
VERCEL_URL=

# Railway (自动配置)
RAILWAY_URL=
```
